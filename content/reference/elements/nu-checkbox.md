---
title: nu-checkbox
type: element
description: A checkbox element with two states.
---

## Usage

Standalone version.

```html
<split/>
<nu-checkbox></nu-checkbox>
<nu-checkbox checked></nu-checkbox>
```

Form version.

```html
<split/>
<nu-form>
  <nu-field>
    <nu-label>Remember me</nu-label>
    <nu-checkbox id="rememberMe"></nu-checkbox>
  </nu-field>
</nu-form>
```
