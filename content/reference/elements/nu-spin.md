## Usage

Use `nu-spin` as a standalone element...

```html
<split/>
<nu-pane>
  <nu-spin></nu-spin>
  <nu-el>Loading...</nu-el>
</nu-pane>
```

... or inject it to action elements using `loading` attribute:

```html
<split/>
<nu-btn special loading>Submit</nu-btn>
```
