## Usage

Change the base font size of the app to `18px`.

```html
<nu-base size="18px"></nu-base>
```

Use can make a responsivee font size by using `responsive` property:

```html
<nu-base responsive="100rem" size="22px|18px"></nu-base>
```

**IMPORTANT**: You can only use a single `nu-base` element in the app.

See an example in our [REPL](!https://numl.design/repl#607553a76cd202483f3a87c6).
